# Privacy Policy — Exam Anxiety Detector

## Data Collection
- **Text Input**: Student text is sent to the API for real-time analysis only.
- **No Storage**: Text is processed in memory and discarded after the response is sent.
- **No PII**: No names, email addresses, IP addresses, or other personal identifiers are collected or stored.

## Data Processing
- Text is analyzed using NLP models (BERT, NLTK, spaCy) running on the server.
- If configured, text may be sent to Google Gemini AI for generating tips (subject to [Google's privacy policy](https://policies.google.com/privacy)).
- All processing is stateless — no session data persists after the browser is closed.

## Data Sharing
- We do not sell or share any user data with third parties.
- Anonymized, aggregated statistics may be used to improve the model.

## User Rights
- Users can close the browser at any time; no data is retained.
- Users must provide consent before submitting text for analysis.

## Contact
For privacy-related inquiries, contact the project maintainers.
